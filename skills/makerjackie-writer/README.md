Maker Jackie 的公众号长文写作 skill。

> 本 skill 改造自数字生命卡兹克的写作 skill，原始版本见：[卡兹克的公众号写作方法论](https://mp.weixin.qq.com/s/AtxGrii_K-nzkwUM9SNhEg?scene=0&subscene=90)

## 安装

```bash
claude code skill add makerjackie/skills/makerjackie-writer
```

## 快速开始

安装后，你可以直接对 Claude 说：

- "帮我写篇公众号文章"
- "用我的风格把这个素材写成长文"
- "续写这篇文章"
- "根据这个 PDF 帮我出稿"

适用场景：公众号长文、深度内容创作、基于素材的文章生成。

不适用：短内容（小红书、推特、朋友圈）、纯标题生成。

## 为什么需要这个 skill？

你有没有遇到过这样的情况：

- 脑子里有很多想法，但一写就卡壳，不知道怎么组织
- 用 AI 写出来的文章总是"AI 味"很重，像机器人在说话
- 想保持自己的风格，但又希望 AI 能帮忙提高效率
- 有一堆素材（PDF、语音转文字、新闻链接），但不知道怎么变成一篇完整的文章

这就是我做这个 skill 的原因。

## 这个 skill 能帮你做什么？

这不是一个简单的"AI 代写工具"。它更像是一个**懂你风格的写作助手**。

它会：

1. **理解你的素材**：不管你给的是 PDF、语音转文字、还是几个散乱的想法，它都能帮你梳理出文章的骨架
2. **保持真人感**：通过四层质检体系，确保输出的文章不是"AI 味"的套话，而是像真人在聊天
3. **符合你的风格**：基于 Jackie 的写作风格（真诚、反焦虑、MVP 思维、动手创造），让文章有温度、有个性
4. **提供创作框架**：从选题判断、内容组织、到最后的质检，给你一套完整的创作流程

## Jackie 的写作风格是什么？

如果用一句话概括：**"一个充满好奇心的创造者，在真诚地分享他的探索和思考。"**

核心特点：

- **真诚第一**：不装、不端、不制造焦虑。可以不完美，但绝不骗人
- **MVP 思维**：强调动手创造，输出是最好的输入
- **反焦虑**：不追求"标准化成功路径"，允许自己慢一点
- **好奇心驱动**：对世界怎么运转充满好奇，喜欢折腾和探索
- **温暖鼓励**：理解那些还在观望、犹豫、内耗的人，想成为"懂你"的人

## 它和其他 AI 写作工具有什么不同？

大多数 AI 写作工具会给你一篇"看起来很完美"的文章，但读起来就是没有灵魂。

这个 skill 的设计理念是：**AI 是辅助，不是替代。**

它会：
- 帮你找证据、找类比、补充背景知识
- 按你确定的角度扩写内容
- 提供结构建议和逻辑梳理

但它不会：
- 替你编造第一手经历（那些必须是真实的）
- 替你想核心创意角度（那个"对，就是这个！"的判断必须是你的）
- 替你表达真实情绪（体感记忆是 AI 给不了的）

## 谁适合用这个 skill？

- 想做自媒体但不知道怎么开始的人
- 有很多想法但写作效率低的创作者
- 希望保持个人风格但又想借助 AI 提效的内容创作者
- 经常需要把素材（会议记录、语音、PDF）整理成文章的人

## 开始使用

安装后，试着给 Claude 一些素材，说"帮我用我的风格写篇文章"。

它会先理解你的素材，判断选题质量，然后按照 Jackie 的风格帮你组织内容。写完后，还会跑一遍四层质检，确保文章有"活人感"。

不用担心一次就完美。这是一个迭代的过程。你可以随时调整、补充真实细节、加入自己的声音。

**记住：AI 是你的助手，不是替代品。最好的内容，永远是人和 AI 协作出来的。**
