makerjackie-writer

MakerJackie 的自媒体写作 Skill

丢给 AI 一个想法、一段素材或一个 PDF，它就能帮你写出一篇有"活人感"的公众号长文。拒绝 AI 味，保持真诚，适合深度内容创作。

核心特性

  • 真诚第一，拒绝制造焦虑
  • MVP 思维，强调动手创造
  • 四层质检体系，确保"活人感"
  • 理解素材，从 PDF、语音、想法到完整文章
  • 保持个人风格，温暖鼓励的表达

使用案例

/makerjackie-writer 帮我写篇公众号文章 主题：AI 时代的创作者如何保持真实
/makerjackie-writer 用我的风格 把这个 PDF 素材写成长文

cloudflare-one

用 CloudFlare 零成本部署你的网站和应用

让 AI 帮你用 Cloudflare 免费额度搭建全栈应用。前端、后端、数据库、存储一条龙，不用自己配服务器。

核心特性

  • 整合 Cloudflare 核心生态服务
  • Workers 无服务器计算平台
  • Pages 静态站点托管
  • D1 数据库、R2 对象存储、KV 键值存储
  • Queues 消息队列、Workers AI 模型推理
  • 符合 Cloudflare 最佳实践

使用案例

/cloudflare-one 部署一个用户预约活动站 前端 HTML+Tailwind 报名信息存入 D1 数据库
/cloudflare-one 创建 API 端点 Workers 处理请求 数据存储在 KV 中
cloudflare-redirector

批量创建短链接和域名跳转

把一堆长链接变成干净的短域名，自动配置跳转和 DNS。适合活动推广、文档分享、让链接看起来更专业。

核心特性

  • 基于 Cloudflare Workers 的极简路由重定向
  • 支持批量配置重定向规则
  • 自动同步 DNS 记录
  • 适用于短链接、活动推广、文档分享
  • 让链接变得干净、专业且易于追踪

使用案例

/cloudflare-redirector 配置规则 docs.example.com → 飞书文档 join.example.com → 微信群二维码
/cloudflare-redirector 创建短链接 go.example.com/meeting → 腾讯会议链接
