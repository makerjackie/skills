Design System / Brand

01mvp-design

当你需要让 AI 生成对外展示的内容（如产品落地页、活动海报、社交媒体图、组件库等）时使用。

它内置了 01MVP 标志性的“黑白高对比、硬核极简、零圆角、工程感排版”设计规范，并携带了官方 Logo SVG。这确保了 AI 产出的设计稿具有高度统一的视觉冲击力，彻底告别那些缺乏辨识度的默认 AI 生成风格。

/01mvp-design 帮我设计一套 01MVP 官网首页 - 包含 Hero 区域 - 包含 3 个功能卡片 - 底部包含一个 CTA 区域
/01mvp-design 请基于 01MVP 极简硬核风格，生成一张产品发布海报。 主标题为“LESS IS MORE”，副标题为“重构工作流”。
Cloudflare / Infra

cloudflare-one

当你需要让 AI 编写后端服务、构建全栈应用或配置基础设施时使用。

它将 Cloudflare 的核心生态（Workers, Pages, D1 数据库, R2 存储, Queues 等）整合在一个技能中。无需你再去四处寻找文档或手动解释部署流程，AI 就能基于此技能生成符合 Cloudflare 最佳实践的代码，并帮你一次性搞定上线。

/cloudflare-one 部署一个用户预约活动站，前端使用 HTML+Tailwind，将用户提交的报名信息（姓名、邮箱）直接存入 D1 数据库。
Cloudflare / Redirect

cloudflare-redirector

当你需要配置短链接、管理域名跳转、或者将长链接（如飞书文档、Notion 页面、问卷星）映射到自定义域名时使用。

它基于 Cloudflare Workers 构建，为你提供了一个极简的路由重定向能力，支持批量配置。无论是活动推广还是内部文档分享，都能让链接变得干净、专业且易于追踪。

/cloudflare-redirector 请帮我配置一下规则： 将 docs.example.com 指向我的飞书公开文档，将 join.example.com 指向我的微信群加群二维码链接。
工作流

如何使用

无需从零解释背景。直接将目标交给 AI，由 Skill 自动加载上下文、资产与执行流程。

01

安装所需技能

将 `01mvp/skills` 整体安装至 Agent 环境，或仅选取特定功能模块。

02

输入目标指令

例如“部署 Pages 站点”或“生成产品海报”，无需重复提供基础信息。

03

自动执行标准流

Skill 将自动调用内置的模板、资产与参考资料，确保产出符合团队规范。
