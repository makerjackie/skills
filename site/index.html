Design System / Brand

01mvp-design

把 01MVP 的 logo、黑白高对比排版、组件规则和模板打包成统一视觉规范，用于海报、名片、UI、网站与各种对外物料。

核心特性

  • 内置 01MVP 官方 Logo SVG 资源
  • 黑白高对比、硬核极简、零圆角设计规范
  • 工程感排版系统，统一视觉冲击力
  • 适用于海报、名片、UI、网站等多种场景

使用案例

/01mvp-design 帮我设计一套 01MVP 官网首页 - 包含 Hero 区域 - 包含 3 个功能卡片 - 底部包含一个 CTA 区域
/01mvp-design 生成一张产品发布海报 主标题"LESS IS MORE" 副标题"重构工作流"
Cloudflare / Infrastructure

cloudflare-one

用一个适合 AI 调用的 Cloudflare Token，把 Workers、Pages、R2、D1、KV、Queues、Browser Rendering、Workers AI 等基础设施串起来。

核心特性

  • 整合 Cloudflare 核心生态服务
  • Workers 无服务器计算平台
  • Pages 静态站点托管
  • D1 数据库、R2 对象存储、KV 键值存储
  • Queues 消息队列、Workers AI 模型推理
  • 符合 Cloudflare 最佳实践

使用案例

/cloudflare-one 部署一个用户预约活动站 前端 HTML+Tailwind 报名信息存入 D1 数据库
/cloudflare-one 创建 API 端点 Workers 处理请求 数据存储在 KV 中
Cloudflare / Redirect

cloudflare-redirector

用 Cloudflare Workers 做批量域名重定向，支持规则编译和 DNS 同步。适合短链接、域名跳转、长链接映射等场景。

核心特性

  • 基于 Cloudflare Workers 的极简路由重定向
  • 支持批量配置重定向规则
  • 自动同步 DNS 记录
  • 适用于短链接、活动推广、文档分享
  • 让链接变得干净、专业且易于追踪

使用案例

/cloudflare-redirector 配置规则 docs.example.com → 飞书文档 join.example.com → 微信群二维码
/cloudflare-redirector 创建短链接 go.example.com/meeting → 腾讯会议链接
