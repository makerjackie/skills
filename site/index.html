makerjackie-writer

MakerJackie 的自媒体写作 Skill

一个帮你把想法、素材、PDF、录音整理成“有活人感”长文的写作 Skill。核心价值就是保留真诚表达、减少 AI 味，并把素材理解、结构组织和语言润色一次做完。

使用案例

/makerjackie-writer 帮我写篇公众号文章 主题：AI 时代的创作者如何保持真实
/makerjackie-writer 用我的风格 把这个 PDF 素材写成长文

使用前提

  • 素材准备：你最好已经有一个想法、提纲、录音、聊天记录或 PDF 素材中的任意一种。
  • 风格输入：如果你希望更像你的语气，给它几段你过去写过的内容会更好。

不用担心具体怎么喂给 AI，真正使用时 Skill 会引导你一步步补齐素材和风格信息。

01mvp-design

一键生成极简硬核黑白风格设计

一个把 01MVP 黑白硬核视觉体系直接交给 AI 的设计 Skill。核心特性是统一高对比视觉、极简排版和现成品牌资源，让海报、名片、页面和 UI 很快落到同一套风格里。

使用案例

/01mvp-design 帮我设计一套 01MVP 官网首页 - 包含 Hero 区域 - 包含 3 个功能卡片 - 底部包含一个 CTA 区域
/01mvp-design 生成一张产品发布海报 主标题"LESS IS MORE" 副标题"重构工作流"

使用前提

  • 场景明确：先告诉它你要做海报、名片、官网页面还是产品 UI。
  • 内容输入：最好准备标题、文案、模块清单或品牌关键词，生成结果会更贴近你的目标。

如果你还没想清楚页面结构，Skill 也会先帮你拆版式，再继续生成视觉方案。

cloudflare-one

用 CloudFlare 零成本部署你的网站和应用

一个让 AI 帮你把前端、后端、数据库和存储都部署到 Cloudflare 生态里的 Skill。核心特性是利用免费额度快速起项目，减少服务器配置负担，并尽量按 Cloudflare 最佳实践落地。

使用案例

/cloudflare-one 部署一个用户预约活动站 前端 HTML+Tailwind 报名信息存入 D1 数据库
/cloudflare-one 创建 API 端点 Workers 处理请求 数据存储在 KV 中

使用前提

  • Cloudflare 账号：你需要有自己的 Cloudflare 账号。
  • API 权限：需要配置对应的 API Token / API Key，Skill 会提示你具体怎么配。

不用提前把所有服务研究明白，真正执行时 Skill 会根据你要部署的类型引导你补配置。

cloudflare-redirector

批量创建短链接和域名跳转

一个帮你批量管理短链接和跳转规则的 Skill。核心特性是把域名、重定向和 DNS 规则串起来，让活动页、文档入口和分享链接更统一、更专业，也更容易维护。

使用案例

/cloudflare-redirector 配置规则 docs.example.com → 飞书文档 join.example.com → 微信群二维码
/cloudflare-redirector 创建短链接 go.example.com/meeting → 腾讯会议链接

使用前提

  • 自己的域名：你需要有一个自己的域名，才能配置短链接和跳转入口。
  • Cloudflare 托管：建议域名已经接入 Cloudflare，方便自动同步 DNS 和 Workers 配置。

这些前提不用你一次性手动配完，Skill 在真正执行时会告诉你下一步该准备什么。
