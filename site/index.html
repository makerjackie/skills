Cloudflare / Infrastructure

cloudflare-one

用一个适合 AI 调用的 Cloudflare Token，把 Workers、Pages、R2、D1、KV、Queues、Browser Rendering、Workers AI 等基础设施串起来。

核心特性

  • 整合 Cloudflare 核心生态服务
  • Workers 无服务器计算平台
  • Pages 静态站点托管
  • D1 数据库、R2 对象存储、KV 键值存储
  • Queues 消息队列、Workers AI 模型推理
  • 符合 Cloudflare 最佳实践

使用案例

/cloudflare-one 部署一个用户预约活动站，前端使用 HTML+Tailwind，将用户提交的报名信息（姓名、邮箱）直接存入 D1 数据库。
/cloudflare-one 创建一个 API 端点，使用 Workers 处理请求，数据存储在 KV 中，并配置 CORS。

Cloudflare / Redirect

cloudflare-redirector

用 Cloudflare Workers 做批量域名重定向，支持规则编译和 DNS 同步。适合短链接、域名跳转、长链接映射等场景。

核心特性

  • 基于 Cloudflare Workers 的极简路由重定向
  • 支持批量配置重定向规则
  • 自动同步 DNS 记录
  • 适用于短链接、活动推广、文档分享
  • 让链接变得干净、专业且易于追踪

使用案例

/cloudflare-redirector 请帮我配置一下规则： 将 docs.example.com 指向我的飞书公开文档， 将 join.example.com 指向我的微信群加群二维码链接。
/cloudflare-redirector 创建短链接系统，将 go.example.com/meeting 重定向到腾讯会议链接。
